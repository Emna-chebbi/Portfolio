# DevOps Portfolio Platform

A cloud-deployed personal portfolio designed to showcase my technical skills, professional experiences, projects, certifications, and achievements as a digital resume.

This project demonstrates DevOps practices by implementing a complete CI/CD pipeline that automatically builds, validates, and deploys changes using GitHub Actions and Netlify.

## ⚙️ CI/CD Workflow

Every update pushed to the `main` branch triggers an automated deployment pipeline:

```
Code Commit
     ↓
GitHub Repository
     ↓
GitHub Actions CI/CD Pipeline
     ↓
Automated Deployment
     ↓
Production Website on Netlify
```

## 🎯 Project Goals

- Demonstrate practical CI/CD implementation
- Automate the deployment lifecycle
- Apply DevOps principles to a real-world project
- Provide a centralized platform showcasing my engineering journey
